Incoming Webhooks for Grafana Incident The Incoming Webhook integration allows you to wire up any third-party system to trigger an Incident. Details of the webhook are attached to the Incident. Get started It takes only a few steps to configure a third-party tool to automatically declare an Incident. Enable Incoming Webhooks In the Grafana Incident web app, an admin can go to Integrations to enable incoming webhooks. Installing the Incoming Webhooks integration will generate a token which you will use to authorize the requests. Go to Alerts & Incidents and under Incident choose Integrations Select the Incoming Webhooks integration Click Install integration Make note of the Token and URL fields
